# 广告混音批量制作工具

把歌曲放入 `input` 目录，把广告音频放入 `ad` 目录，运行程序后会自动批量制作广告版音频，并保存到 `output` 目录。

## 目录结构

```text
.
├─ 广告混音工具.exe
├─ batch_ad_mix.py
├─ input
│  ├─ 歌曲1.mp3
│  └─ 歌曲2.mp3
├─ ad
│  ├─ 广告1.wav
│  └─ 广告2.mp3
└─ output
   ├─ 歌曲1_广告1_广告版_105s.mp3
   └─ batch_report.json
```

## 使用方法

### 方法一：直接双击 exe

双击 `广告混音工具.exe` 即可开始处理。

程序会自动：

- 读取 `input` 里的所有歌曲
- 读取 `ad` 里的所有广告音频
- 按 `歌曲 × 广告` 批量生成广告版音频
- 保存到 `output`
- 生成处理报告 `output\batch_report.json`

### 方法二：命令行运行 exe

```powershell
.\广告混音工具.exe
```

## 广告 TTS 合成工具

如果需要把广告文案合成为语音，可以使用：

```text
https://tts.lp0.cc/
```

建议流程：

- 打开 `https://tts.lp0.cc/`
- 输入广告文案并合成语音
- 下载生成的音频文件
- 把音频放入 `ad` 目录
- 运行 `广告混音工具.exe` 批量生成广告版歌曲

## 默认制作参数

- 广告插入位置：在歌曲 `20s~40s` 范围内自动找低能量、较平稳的位置
- 广告期间背景音乐音量：`30%`
- 广告音频音量：`230%`
- 目标参考时长：`120s`
- 结尾策略：不硬卡参考时长，优先按音乐节奏找自然结尾
- 成品最长：不超过原歌曲时长的 `60%`
- 结尾淡出：`3s`

## 常用参数

调整广告开始搜索范围：

```powershell
.\广告混音工具.exe --search_start_sec 20 --search_end_sec 40
```

调整广告音量：

```powershell
.\广告混音工具.exe --ad_volume 2.6
```

调整广告期间背景音乐音量：

```powershell
.\广告混音工具.exe --duck_volume 0.25
```

调整成品最长比例，例如最多保留原歌曲 50%：

```powershell
.\广告混音工具.exe --max_duration_ratio 0.5
```

只使用某一个广告文件：

```powershell
.\广告混音工具.exe --ad ad\ad.wav
```

## 支持格式

歌曲和广告都支持：

```text
mp3, wav, flac, m4a, ogg, aac
```

## 输出说明

输出文件命名格式：

```text
歌曲名_广告名_广告版_时长s.mp3
```

处理报告：

```text
output\batch_report.json
```

报告里包含每首歌的广告开始时间、结尾淡出位置、实际成品时长和处理状态。

## 注意事项

- `input` 和 `ad` 目录必须和 exe 放在同一层。
- 如果广告太小，增大 `--ad_volume`，例如 `2.6`。
- 如果背景音乐仍然盖住广告，降低 `--duck_volume`，例如 `0.2`。
- 如果成品太长，降低 `--max_duration_ratio`，例如 `0.5`。
- 第一次运行可能稍慢，因为程序需要分析音频节奏。
